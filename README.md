个人向的算法竞赛代码模板，使用 `latex` 编写。

## 使用字体

- `Source Han Serif CN`（思源宋体 CN）
- `WenQuanYi Micro Hei`（文泉驿微米黑）
- `JetBrains Mono NL`

## 构建

```sh
make
```
